## 📌 Ce que j’ai fait dans ce chapitre
Dans cette partie, j’ai travaillé sur les modèles de régression pour prédire des valeurs continues.

### 🔹 Simple Linear Regression
* Relation entre une variable et la target

### 🔹 Multiple Linear Regression
* Plusieurs variables pour améliorer la prédiction

### 🔹 Polynomial Regression
* Modèle non linéaire avec transformation des features

### 🔹 SVR (Support Vector Regression)
* Modèle plus robuste pour capturer des relations complexes

### 🔹 Decision Tree Regression
* Modèle basé sur des décisions (tree structure)

### 🔹 Random Forest Regression
* Ensemble de plusieurs arbres pour améliorer la précision

---

## 🎯 Objectif
Comprendre les différents modèles de régression et comment les utiliser selon le problème.

## 💡 Conclusion
Chaque modèle a ses avantages, le choix dépend des données et du problème.
