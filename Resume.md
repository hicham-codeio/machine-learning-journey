### 🔹 Foundation
🔹 Imputation : تعمير القيم لي ناقصين
🔹 Encoding : تحويل الكلمات لأرقام
🔹 ColumnTransformer : كل نوع ديال data كنعالجو بطريقة مختلفة
🔹 Pipeline : تنظيم الخدمة وربط المراحل
🔹 Data Balancing : تصحيح عدم توازن الداتا

### 🔹 Regression
🔹 Simple Linear Regression : علاقة بسيطة بين واحد المتغير و النتيجة
🔹 Multiple Linear Regression : بزاف ديال المتغيرات كيعطيو نتيجة
🔹 Polynomial Regression : علاقة معقدة شوية ماشي خط مستقيم
🔹 SVR : model قوي كيتعامل مع العلاقات الصعيبة
🔹 Decision Tree Regression : قرارات على شكل شجرة
🔹 Random Forest Regression : مجموعة ديال الأشجار كيعطيو نتيجة أدق

### 🔹 Classification
🔹 Logistic Regression : كيتوقع 0 ولا 1
🔹 K-NN : كيشوف أقرب neighbors ويقرر
🔹 SVM : كيقسم data بأفضل خط ممكن
🔹 Kernel SVM : كيتعامل مع data لي ماشي مستقيمة
🔹 Naive Bayes : كيعتمد على الاحتمالات
🔹 Decision Tree : قرارات بسيطة على شكل شجرة
🔹 Random Forest : عدة أشجار باش نحسنو النتيجة

### 🔹 Clustering
🔹 K-Means : كيقسم data لمجموعات متشابهة
🔹 Hierarchical : كيدير clustering على شكل شجرة

### 🔹 Association Rule Learning
🔹 Apriori : كيلقى الحوايج لي كيتباعو مع بعضهم
🔹 Eclat : نفس الفكرة ولكن بطريقة أسرع

### 🔹 Reinforcement Learning
🔹 Q-Learning : كيتعلم بالتجربة والمكافأة
🔹 Exploration vs Exploitation : يوازن بين يجرب ويتعلم

### 🔹 NLP
🔹 Text Cleaning : تنظيف النص من الحوايج الزايدين
🔹 Bag of Words : تحويل النص لأرقام
🔹 TF-IDF : كيعرف الكلمات المهمة
🔹 Sentiment Analysis : واش الرأي إيجابي ولا سلبي

### 🔹 ANN
🔹 ANN : شبكة بحال الدماغ كتتعلم من data
🔹 Forward Propagation : تمرير data فالشبكة
🔹 Backpropagation : تصحيح الأخطاء

### 🔹 CNN
🔹 CNN : شبكة خاصة بالصور
🔹 Feature Extraction : استخراج المعلومات من الصورة
🔹 Image Classification : تصنيف الصور

### 🔹 Dimensionality Reduction
🔹 PCA : نقصو عدد المتغيرات ونحافظو على المهم
🔹 LDA : نحسنو الفرق بين الكلاسات

### 🔹 Model Selection
🔹 Cross Validation : نختبرو model بطريقة أدق
🔹 Grid Search : نقلبو على أحسن paramètres
🔹 Metrics : نقيسو الأداء ديال model
