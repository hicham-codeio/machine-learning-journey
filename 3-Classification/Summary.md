## 📌 Ce que j’ai fait dans ce chapitre
Dans cette partie, j’ai travaillé sur les modèles de classification pour prédire des classes (0/1 ou multi-classes).

### 🔹 Logistic Regression
* Modèle simple pour classification binaire

### 🔹 K-Nearest Neighbors (K-NN)
* Classification basée sur les voisins les plus proches

### 🔹 Support Vector Machine (SVM)
* Trouver la meilleure séparation entre les classes

### 🔹 Kernel SVM
* Gérer les cas non linéaires avec kernels

### 🔹 Naive Bayes
* Basé sur les probabilités (Bayes theorem)

### 🔹 Decision Tree Classification
* Modèle basé sur des règles (arbre de décision)

### 🔹 Random Forest Classification
* Plusieurs arbres pour améliorer la précision

## 🎯 Objectif
Comprendre les différents modèles de classification et leur utilisation.

## 💡 Conclusion
Le choix du modèle dépend du dataset et du type de problème.
